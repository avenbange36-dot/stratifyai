# StratifyAI Enterprise

StratifyAI Enterprise is a SaaS foundation for an AI consulting platform. It includes a marketing website, account creation, login, workspace management, AI readiness assessment, recommendation engine, PDF reports, Stripe payment foundation, enterprise SSO placeholders, audit-log infrastructure, and production-readiness documentation.

## Local start

Install Python 3.11+ and Node.js LTS, then run the START file for your operating system:

- Windows: `START_WINDOWS.bat`
- Mac: `START_MAC.command`
- Linux: `START_LINUX.sh`

Open `http://localhost:5173`.

## What is included

- React frontend marketing site and SaaS dashboard.
- FastAPI backend.
- Account creation and login.
- Organization/workspace management.
- AI readiness scoring and roadmap generation.
- PDF report export.
- Stripe Checkout foundation and webhook endpoint.
- Audit log model and endpoints.
- Enterprise SSO configuration placeholders.
- Security control tracking endpoint.
- Logo and platform name.
- Website download page with downloadable package.
- Production launch checklist.
- Privacy Policy, Terms, and DPA templates.
- SOC 2 readiness plan.
- Penetration test plan.
- Vercel/Render/Supabase and AWS deployment guides.

## Important

This is a strong SaaS foundation, not a finished enterprise-certified product. You still need your own hosting accounts, domain, Stripe account, legal review, SSO vendor, security testing, monitoring, backups, and compliance audit before selling to Fortune 500 customers.
